FLIR R-JPEG Camera Model

* FLIR
* FLIR AX8
* FLIR B60
* FLIR E40
* FLIR T640

DJI R-JPEG Camera Model

* DJI H20T
* DJI XT2
* DJI XTR
* DJI XTS

DJI R-JPEG Camera Model DTAT3.0

* DJI M2EA / DJI MAVIC2-ENTERPRISE-ADVANCED / "御"2 行业进阶版
* DJI H20N
* DJI M3T / DJI MAVIC3
* DJI M30T
